# "Sprites" usando Pygame
Este é um simples jogo em Python utilizando a biblioteca Pygame, chamado "Sprites". No jogo, você controla um personagem (sapo) que pode executar uma animação de ataque ao pressionar uma tecla. Este Readme irá fornecer informações básicas sobre como executar o jogo e entender sua estrutura.
# Controles
Pressione qualquer tecla para acionar a animação de ataque do sapo.

# Estrutura do Código
O código do jogo "Sprites" segue uma estrutura simples:
Importação das bibliotecas necessárias, incluindo o Pygame.
Inicialização do Pygame e configuração da janela.
Definição de uma classe Sapo que representa o personagem jogável.
Criação de uma tela, sprites e grupos de sprites.
Loop principal do jogo que controla a animação do sapo e a interação do jogador.

# Personalização
Você pode personalizar este jogo da seguinte maneira:
Alterar as imagens dos sprites do sapo na pasta 'sprites'.
Ajustar a velocidade da animação ou a taxa de quadros por segundo (FPS).
Adicionar mais recursos, como música de fundo, efeitos sonoros ou inimigos.
Créditos
Este código foi criado como um exemplo simples de jogo em Pygame e pode ser usado como ponto de partida para projetos mais complexos. Os sprites do sapo são apenas um exemplo e podem ser substituídos por recursos personalizados.

Divirta-se jogando e explorando o código!
